## Compilation

Install dependencies (keeping the configured one)

```sh
npm ci
```

Install dependencies (updating to the latest available versions)

```sh
npm install
```

Build for development

```sh
npm run dev
```

Build for production

```sh
npm run build
```

Preview the app built for production

```sh
npm run preview
```
